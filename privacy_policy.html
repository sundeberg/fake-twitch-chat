Privacy Policy

Fake Twitch Chat - Last updated: March 2026

Fake Twitch Chat is a browser extension that generates AI-powered fake Twitch-style chat reactions to whatever page you are viewing. This policy explains what data the extension handles and how it is used.

Data stored locally on your device

The following is stored in chrome.storage.local, which is sandboxed to the extension and never transmitted to the developer:

  • Your Anthropic and/or OpenAI API keys
  • Your chosen AI provider
  • Your chosen streamer name
  • Your custom prompt (if set)
  • Per-site settings (chat mode, tier, intensity, font size, consent approvals)
  • Sidebar width preference and theme preference
  • Context size preference
  • Usage statistics (estimated cost, API call count, token count)
  • Whitelist mode toggle and approved hostnames
  • Custom blocked sites list
  • A log of the last 20 successful AI fetches (page title, extracted context, mode, tier)
  • A cache of BetterTTV global emote IDs (refreshed every 24 hours)

None of this data is collected by or transmitted to the developer.

Data sent to third-party services

Anthropic API

When you use the Standard or Turbo tier with Anthropic selected as your provider, the extension sends requests directly from your browser to api.anthropic.com using your own API key. Each request includes:

  • A summary of the current page (title, description, and site-specific content such as a YouTube video title, Reddit post body, or GitHub repository description)
  • Your selected chat mode and intensity
  • Your streamer name (if set)
  • Your custom prompt (if set)
  • The last few chat messages as context
  • Any message you typed into the chat input

This data is processed by Anthropic under their Privacy Policy and Usage Policy. The developer of this extension does not receive, store, or have access to any of these requests.

OpenAI API

When you use the Standard or Turbo tier with OpenAI selected as your provider, the extension sends the same set of data described above directly from your browser to api.openai.com using your own API key. This data is processed by OpenAI under their Privacy Policy. The developer of this extension does not receive, store, or have access to any of these requests.

BetterTTV API

Once per 24-hour period, the extension fetches the global emote list from the BetterTTV API (api.betterttv.net) to resolve emote names to images. No personal data is sent in this request.

Data the developer does not collect

The developer collects no data whatsoever. There are no analytics, no telemetry, no crash reports, and no backend server associated with this extension.

Permissions used

  • storage - to save your settings and API key locally
  • activeTab / scripting - to inject the chat sidebar into the current page
  • host permissions - to read page context from the sites you visit, and to make API calls to api.anthropic.com and api.betterttv.net

Children

This extension is not directed at children under 13 and does not knowingly collect any information from children.

Changes to this policy
